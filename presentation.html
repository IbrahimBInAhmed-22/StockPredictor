System Architecture

1. Data Collection

Downloads 10 years of stock data from Yahoo Finance API

ticker = yf.Ticker('AAPL')
data = ticker.history(start=start_date, end=end_date)

2. Technical Analysis

Calculates moving averages (MA20, MA50) and prepares data

data['MA20'] = data['Close'].rolling(window=20).mean()
data['MA50'] = data['Close'].rolling(window=50).mean()

3. Chart Generation

Creates candlestick charts with 30-day windows and volume indicators

4. Intelligent Labeling

Labels data using established methodologies from online trading scripts with proper validation. Based on future price movement: Buy (10%+ gain), Sell (10%+ loss), Hold (otherwise)

5. CNN Training

Trains CNN model with data augmentation and 70/30 train-test split

model = create_cnn_model()
model.fc = nn.Sequential(
nn.Dropout(0.5),
nn.Linear(num_features, 512),
nn.ReLU(),
nn.Dropout(0.3),
nn.Linear(512, 3)
)

How It Works

Step 1: Historical Data

The system downloads comprehensive historical stock data including Open, High, Low, Close prices and Volume

Step 2: Visual Representation

Converts numerical data into visual candlestick charts that capture market patterns and trends

Step 3: Future Analysis

Uses validated labeling scripts from financial analysis repositories to analyze the next 10 days and determine if significant price movements occur, creating ground truth labels

Step 4: Deep Learning

CNN architecture learns to recognize visual patterns in charts that correlate with future price movements

Step 5: Prediction

The trained model can analyze new charts and predict whether to Buy, Sell, or Hold
